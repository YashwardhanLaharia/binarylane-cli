# BinaryLane CLI

A lightweight, interactive terminal-based interface (TUI) to manage your BinaryLane servers and monitor performance via the [BinaryLane](https://www.binarylane.com.au/) API.

## Features

- **Account Information:** View your current account configuration.
- **Server Management:** List all servers, view detailed specifications, and check network configurations.
- **Server Actions:** Perform critical tasks like reboots, power-ons, and power-offs directly from the terminal.
- **Performance & Usage Tracking:** Monitor real-time CPU, RAM, disk I/O, and current data transfer limits.

## Prerequisites

- Python 3.8+
- A BinaryLane API Token (generated from your BinaryLane dashboard)
